# Weekplot
Reads & plots your weekly schedule from a YAML or a simple text file.

![Schedule](Schedule.png)

## Install Dependencies
``` sh
pip3 install -r requirements.txt
```

## Usage
``` sh
# .txt input
python3 weekplot.py events.txt

# .yml input
python3 weekplot.py events.yml
```

#### Example input files
 - [text](events.txt)
 - [YAML](events.yml)
